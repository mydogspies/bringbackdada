What is bri+gba*kDADA?
bri+gba*kDADA (pron: "bringbackdada") is a photography project by Berlin-based visual artist Silenceisgrand (SiG for short). Afters years of urban imagery it was time to explore something more organic and this site is the fruition of a journey into the qualities of the human body as both a canvas and subject. SiG brings here his usual flair for lighting as something of a dialogue with the subject and composition as an actively guiding path through each image. This project is pretty fresh and for now an open canvas for ideas without setting any particular limits.
Model?
If you would find it interesting to model for bri+gba*kDADA then feel free to contact SiG using this form. A few guidelines beforehand though;
* Most work is on a TFP (time for prints/file) basis simply because this is not a commercial project and we have no budgets.
* You have to be available in Berlin (Germany) for both the photo session/s and for an initial personal meeting beforehand to discuss the details, expectations and workflow.
* All models are expected to sign a model release form. Please read it before contacting!
* Everyone on the set works ethically (here is a great take on that, albeit not directly related to the topics here) but the set is not open for chaperons. If you can't work alone directly with the photographer and/or other creative staff then this project is not for you. We simply feel it interrupts the creative flow in a negative way.
* Please include links to a portfolio of your past work. If you are completely new to photography then please send a link to at least one image with your face only and one with your full body. Quality does not matter, it's just so that we have something to go on when talking for the first time.
